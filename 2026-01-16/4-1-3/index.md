# 404와 500의 차이

## 404

- 상태 코드 404 Not Found

## 500

- 상태 코드 500 Internal Server ERror

## 상태 코드 4xx와 5xx의 차이점

- 400 : Bad Request 잘못된 요청 ex) 요청 형식 오류
- 401 : Unauthorized 인증 필요 ex) 로그인 필요
- 403 : Forbidden 접근 금지 ex) 권한 없음
- 404 : Not Found 없음 ex) 리소스가 존재하지 않음
- 500 : Internal Server Error 서버 오류 ex) 서버 내부 오류
- 502 : bad gateway 게이트웨이 에러 ex) 프록시 뒤 서버 문제
- 503 : Service Unavailable 서비스 불가 ex) 서버 과부하/ 점검 중
- 504 : Gateway Timeout 게이트웨이 타임아웃 ex) 백엔드 서버 무응답, 네트워크 지연
